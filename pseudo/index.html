CSS Pseudo-Classes

A CSS pseudo-class is a CSS keyword added to an element that let's you control the style of a specific state of the element, such as selected, hover, or nth-type/nth-child.

Commonly used Pseudo-Classes

:hover

:hover is used to change the style of an element when the user hovers over it with their cursor.

Example:

:active

:active is used to change the style of an element when it's being actively interacted with, usually a click before releasing the mouse.

Example:

:focus

:focus is used to style whatever has the current keyboard focus, generally used for links, text areas, and input fields.

Example:

:link and :visited

:link is used to style links that haven't been visited yet and :visited is used to style links that have been visited.
